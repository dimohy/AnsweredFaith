# 질문하는 믿음, 답하시는 하나님

**출처: 지성이 성경에 묻다, 이원재 / "질문하는 믿음, 답하시는 하나님" 정세일 편저**

---

> *"지옥은 영원한가?", "구원은 한 번뿐인가?"*

이 책은 이러한 신학적 질문에서 출발합니다. 전통적 교리를 전면 부정하지 않으면서, 성경 원문과 하나님의 성품에 근거하여 함께 탐구하는 여정입니다.

---

## 이 책에서 다루는 질문들

- 성경을 어떤 관점으로 읽어야 할까?
- 교리는 하나님의 성품을 제대로 반영하고 있는가?
- 지옥에서도 구원의 가능성이 있을까?
- 천국에서 다시 죄를 지을 수 있을까?
- 하나님의 궁극적인 구속 계획은 무엇인가?

---

## 읽는 방법

왼쪽 사이드바에서 목차를 확인하고, 관심 있는 주제부터 읽으셔도 됩니다.

순서대로 읽으시면 저자의 논지를 따라가기 쉽습니다.
